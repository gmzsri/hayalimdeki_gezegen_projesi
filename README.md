## 📖 Proje Hakkında

**Hayalimdeki Gezegen**, çocukların eğlenerek öğrenmesini sağlayan kapsamlı bir eğitici oyunlar platformudur. Bu proje, modern web teknolojileri ve Flutter framework'ü kullanılarak geliştirilmiştir. Çocukların bilişsel gelişimini destekleyen interaktif oyunlar, renkli arayüzler ve eğitici içerikler sunar.

### 🎯 Hedef Kitle
- **Yaş Grubu:** 5-12 yaş arası çocuklar
- **Kullanıcılar:** İlkokul öğrencileri, ebeveynler ve eğitimciler
- **Platform:** Web tarayıcıları ve mobil cihazlar

## ✨ Özellikler

### 🧠 Eğitici İçerik
- **Bilişsel Gelişim:** Hafıza, dikkat ve problem çözme becerilerini geliştiren oyunlar
- **Matematik Becerileri:** Toplama, çıkarma, çarpma işlemleri ile matematik pratiği
- **Görsel Algı:** Renk, şekil ve desen eşleştirme aktiviteleri
- **Mantık Oyunları:** Strateji ve planlama gerektiren zeka oyunları

### 🎨 Kullanıcı Deneyimi
- **Renkli Tasarım:** Çocukların ilgisini çekecek canlı renkler ve animasyonlar
- **Sezgisel Arayüz:** Kolay kullanım için özel olarak tasarlanmış navigasyon
- **Responsive Tasarım:** Tüm cihazlarda mükemmel görünüm
- **Erişilebilirlik:** Farklı ihtiyaçlara sahip çocuklar için uygun tasarım

### 🏆 Motivasyon Sistemi
- **Puan Sistemi:** Başarıları ödüllendiren puan mekanizması
- **Seviye İlerlemesi:** Zorluk seviyesi artan oyunlar
- **Başarı Rozetleri:** Motivasyonu artıran başarı işaretleri
- **İlerleme Takibi:** Gelişimi gösteren detaylı raporlar

### 🔒 Güvenlik ve Güvenilirlik
- **Reklamsız İçerik:** Çocuklar için tamamen güvenli deneyim
- **Veri Gizliliği:** Kişisel bilgilerin korunması
- **İçerik Filtreleme:** Uygun olmayan içeriklerin filtrelenmesi
- **Ebeveyn Kontrolü:** Ailelerin kontrol edebileceği ayarlar

## 🎮 Oyunlar

### 🧩 Hafıza Oyunu
**Amaç:** Hafıza ve görsel algı becerilerini geliştirmek

**Özellikler:**
- Farklı zorluk seviyeleri (Kolay, Orta, Zor)
- Renkli ve eğlenceli kart tasarımları
- Süre sınırlı ve süresiz modlar
- Puan sistemi ve en yüksek skor takibi
- Ses efektleri ve animasyonlar

**Geliştirilen Beceriler:**
- Kısa süreli hafıza
- Görsel algı
- Dikkat konsantrasyonu
- Hızlı düşünme

### 🧮 Matematik Oyunu
**Amaç:** Matematik becerilerini eğlenceli bir şekilde geliştirmek

**Özellikler:**
- Toplama, çıkarma, çarpma işlemleri
- Zorluk seviyelerine göre ayarlanabilir içerik
- Anlık geri bildirim sistemi
- Doğru/yanlış istatistikleri
- Başarı rozetleri ve ödüller

**Geliştirilen Beceriler:**
- Matematiksel düşünme
- Hızlı hesaplama
- Problem çözme
- Mantık yürütme

## 🚀 Kurulum ve Çalıştırma

### Gereksinimler
- Flutter SDK (3.0.0 veya üzeri)
- Dart SDK (2.17.0 veya üzeri)
- Android Studio / VS Code
- Git

### Kurulum Adımları

1. **Projeyi Klonlayın**
   ```bash
   git clone https://github.com/gmzsri/hayalimdeki_gezegen_projesi.git
   cd hayalimdeki_gezegen_projesi
   ```

2. **Bağımlılıkları Yükleyin**
   ```bash
   flutter pub get
   ```

3. **Uygulamayı Çalıştırın**
   ```bash
   # Web için
   flutter run -d chrome
   
   # Android için
   flutter run -d android
   
   # iOS için
   flutter run -d ios
   ```

### Web Uygulamasını Açma
1. `docs/index.html` dosyasını web tarayıcınızda açın
2. Veya `web/index.html` dosyasını Flutter web build'i ile çalıştırın
